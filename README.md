# Customer Churn Analysis

## Project Overview
This project analyzes customer churn behavior and identifies factors influencing customer retention.

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Tableau

## Key Insights
- Customers with international plans showed higher churn rates.
- Frequent customer service calls were associated with increased churn.
- Churned customers had higher average day charges.
